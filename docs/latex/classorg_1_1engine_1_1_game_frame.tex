\doxysubsection{Detailed Description}
The Game\+Frame class is a part of the Game Engine system that handles switching and displaying appropriate game states. 

It extends JFrame to use Swing for displaying a game window and implements State\+Change\+Listener to receive requests from the states to switch to a different state. \begin{DoxyAuthor}{Author}
